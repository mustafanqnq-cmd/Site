# بوت التواصل (Tython Support Bot)

بوت تيليجرام بسيط: أي شخص يكتب له، تنوصلك رسالته، وإذا رديت عليها (Reply) داخل تيليجرام يوصل ردك للشخص نفسه.
لا يحتاج قاعدة بيانات.

## 1. الرفع على GitHub
1. أنشئ Repo جديد على GitHub.
2. ارفع هذي الملفات الثلاثة: `bot.py` و `requirements.txt` و `Procfile`.

## 2. الرفع على Railway
1. سجل دخول على https://railway.app
2. New Project → Deploy from GitHub repo → اختر الريبو.
3. روح لـ Variables وضيف:
   - `BOT_TOKEN` = توكن البوت من BotFather
   - `ADMIN_ID` = آيدي حسابك على تيليجرام (رقم، تاخذه من @userinfobot)
4. روح لـ Settings → Deploy وتأكد إن نوع الخدمة "Worker" (مال Procfile) مو Web، لأن البوت مايحتاج بورت.
5. اضغط Deploy وانتظر، وراح يبدأ البوت يشتغل فورًا (Polling).

## ملاحظات
- لو أعدت تشغيل البوت (Restart)، الرسائل المنتظرة رد قبل الريستارت ما راح يقدر يربطها بصاحبها (لأن التخزين بالذاكرة بس)، لكن أي رسالة جديدة تشتغل عادي.
- تقدر بأي وقت تطلب مني أضيف عليه قاعدة بيانات لو احتجت يحفظ سجل المحادثات بشكل دائم.
